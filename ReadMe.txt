Space Invaders but as a Web Application and with a Account System.
Scores get saved in a db and the top 10 scores are displayed.

Inside the /dist folder:
space_invaders_rewebed-1.0.tar.gz

To install (from dist folder):
pip install space_invaders_rewebed-1.0.tar.gz

To run:
run_space_invaders_rewebed
